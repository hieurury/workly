# Workly

**"Không có việc gì khó, chỉ sợ bạn mất streak."**

Workly là một ứng dụng di động hỗ trợ quản lý công việc và tính toán thu nhập cá nhân theo ngày/chu kỳ một cách thông minh, tối ưu và giao diện tuyệt đẹp. App được xây dựng bằng Flutter.

## Các tính năng chính
- **Quản lý Công Việc**: Thêm mới các công việc với cấu hình lương cơ bản, lương tăng ca, trợ cấp, cấu hình ngày nghỉ cuối tuần.
- **Điểm danh Hằng Ngày**: Theo dõi tiến độ làm việc, ghi nhận đi làm, tăng ca, hoặc nghỉ làm.
- **Dự đoán Thu Nhập**: Tự động tính toán tổng lương chu kỳ dựa vào số ngày đã đi làm, cũng như dự đoán tổng thu nhập nếu bạn giữ nguyên streak.
- **Biểu đồ Trực Quan**: Cung cấp biểu đồ tròn, biểu đồ đường để theo dõi cơ cấu lương và tiến độ.
- **Giao diện Tối/Sáng**: Hỗ trợ Dark Mode và Light Mode sang trọng, mượt mà.
- **Sao chép Chu Kỳ**: Tự động nhân bản công việc cũ sang chu kỳ mới khi nhận lương xong, không cần cấu hình lại.

## Cài đặt và Chạy thử
1. Yêu cầu [Flutter SDK](https://docs.flutter.dev/get-started/install) phiên bản mới nhất.
2. Clone dự án:
   ```bash
   git clone https://github.com/hieurury/workly.git
   ```
3. Truy cập thư mục dự án:
   ```bash
   cd workly
   ```
4. Cài đặt các dependencies:
   ```bash
   flutter pub get
   ```
5. Chạy ứng dụng trên thiết bị ảo/thật:
   ```bash
   flutter run
   ```
